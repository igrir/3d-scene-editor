🏠 3D Primitive Builder

Browser-based 3D scene editor & embeddable library — built with Three.js

📦 GitHub 📖 Embed Docs ⬇️ Download JS

✨ Features

  • 15 primitives — Box, Sphere, Cylinder, Cone, Torus, Half/Quarter Donut, Half/Quarter Ball, Bowl, Triangles, Plane, Image, Bookshelf
  • Gizmo transforms — Move, rotate, scale with Simple or Advanced gizmo
  • Color system — HSV picker, swatches, paint bucket, eyedropper, sync with selection
  • Multi-select — Ctrl+Click, Rectangle select, group transforms
  • Undo/Redo — Full history with keyboard shortcuts
  • World controls — Sun light, ambient, background, grid color/opacity, shadows, SSAO
  • Save/Load — localStorage saves with thumbnails, JSON export/import
  • Sample projects — Built-in scenes: Rumah, Gedung, Gunung, Bunga, Rak Buku

🌐 One-Line Embed

<script src="dist/3d-primitive-builder.umd.js"></script>
<div data-primitive-editor='{"height":500}'></div>

See embedding docs for full API.

🧰 Tech Stack

  • Three.js r157 — WebGL rendering
  • vanilla-picker — Color picker UI
  • Vite — Build tool
  • Vanilla JS — No framework
