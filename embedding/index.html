🌐 Embedding 3D Primitive Builder

Include the full 3D scene editor on any website with a single <script> tag. No build tools, no framework, no server required.

Quick Start

<div id="editor" style="width:100%;height:500px"></div>
<script src="../dist/3d-primitive-builder.umd.js"></script>
<script>
  PrimitiveBuilder.create('#editor', { height: 500 }).init();
</script>

Zero-JS Embed

<div data-primitive-editor='{"height":500}'></div>
<script src="../dist/3d-primitive-builder.umd.js"></script>

API Reference

MethodDescription
.init()Initialize the 3D scene (async)
.addPrimitive(type)Add an object (box, sphere, etc.)
.loadScene(data)Load scene from JSON
.getSceneData()Export scene as JSON
.clearScene()Remove all objects
.setBackgroundColor(hex)Change background
.destroy()Clean up

Example

const editor = await PrimitiveBuilder.create('#editor', {
  height: 520,
}).init();

editor.addPrimitive('box');
editor.addPrimitive('sphere');

const data = editor.getSceneData();
console.log(data);

// Clean up when done
// editor.destroy();
